src.setup.encode_string_list(x)¶
Summary

Take a list of strings x and encode its values to a list of integers.

Parameters

x – list of strings

Return y

list of integers that are codes for x. x is sorted before being encoded.

Example

x = [‘cat’, ‘dog’, ‘book’, ‘pencil’, ‘dog’, ‘book’]

y = [1, 2, 0, 3, 2, 0]
